# Basic Expense Tracker 💸

A simple console-based expense tracking application written in C++.

## Features

* Add income transactions
* Add expense transactions
* Sort transactions by date
* Display transaction history
* Save data to CSV file
* Set saving target
* Set maximum expense warning

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL Vector
* File Handling (CSV Export)

## Project Structure

* `Transaction` → stores transaction data
* `Expense_manage` → manages business logic
* `File_Manage` → handles file export

## Purpose

This project was built to:

* practice OOP in C++
* apply STL and sorting
* create a useful daily-life application
